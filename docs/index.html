Icons

  • Images
    • Big-file
    • Not customizable
  • Fonts
    • Single controllable color
    • Tricky to position
    • Can be blocked
    • Not accessible
    • Anti-aliased
    • Scalable
  • SVG
    • Fully customizable
    • Animation-friendly
    • Responsive

Animation

  • A sense of progress
    • "We're moving forward/backwards"
  • A sense that something is happening
    • It cuts down on perceived wait time
    • e.g. loaders
  • A sense of orientation
    • "where did it go?"
    • retain context
  • A call to action
  • Clarifying / explanatory animation
  • Decoration / entertainment-sake
